This plugin disables the WordPress emoji functionality.


== Description ==

This plugin disables the WordPress emoji functionality.


Note: Emoticons will still work and emoji's will still work in browsers which have built in support for them. This plugin simply removes the extra code bloat used to add support for emoji's in older browsers.


== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'CS Disable Emojis'
3. Activate CS Disable Emojis from your Plugins page.

= From WordPress.org =

1. Download CS Disable Emojis.
2. Upload the 'cs-disable-emojis' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate CS Disable Emojis from your Plugins page.
